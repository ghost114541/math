# 第一阶段：页面结构草图

## 通用页面布局

```text
+------------------------------------------------------------+
| 左侧导航栏                                                   |
+----------------------+-------------------------------------+
| 控制面板              | 可视化区域                           |
|                      |                                     |
| - 参数调节            | - 几何示意图 / 3D 场景                |
| - 当前模式            | - 高亮点、射线、平面                  |
| - 关键数值            | - 当前计算结果                        |
+----------------------+-------------------------------------+
| 矩阵和公式展示区域                                          |
| - 源矩阵                                                    |
| - 合成矩阵                                                  |
| - 关键公式 / 算法状态                                       |
+------------------------------------------------------------+
```

## 页面 1：一点透视投影

- 控制面板：
  - 投影平面距离 `d`；
  - `x_view`、`y_view`、`z_view`；
  - 坐标约定说明。
- 可视化区域：
  - 位于原点的相机；
  - 投影平面 `z = -d`；
  - 从相机穿过点的投影射线；
  - 投影平面上的投影点。
- 公式/矩阵区域：
  - 几何函数公式；
  - 一点透视的教学用齐次矩阵；
  - 标准透视投影矩阵。

## 页面 2：ModelViewMatrix

- 控制面板：
  - 模型旋转；
  - 模型平移；
  - 相机位置。
- 可视化区域：
  - 模型原点；
  - 相机位置；
  - 示例顶点；
  - local -> world -> view 的坐标变化链。
- 公式/矩阵区域：
  - `M_model`；
  - `M_view`；
  - `M_modelView = M_view * M_model`。

## 页面 3：3D Clipping

- 控制面板：
  - 视场角；
  - near 平面；
  - far 平面；
  - 图元偏移量。
- 可视化区域：
  - view space -> clip space -> clipping -> perspective divide -> NDC -> screen；
  - 原始图元；
  - 裁剪后图元；
  - inside/outside/intersecting 状态。
- 公式/算法区域：
  - OpenGL/WebGL clip-space 不等式；
  - 线段裁剪；
  - 三角形裁剪。
